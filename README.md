# NutiFuel - Supplement Store Website
NutriFuel is a modern marketing website for a nutritional supplements brand. It showcases high-quality products designed to support fitness, health, and an active lifestyle.

## Features
- Responsive design
- Modern and clean user interface
- Product showcase
- About section
- Contact form
- Smooth navigation

## Technologies Used
- HTML5
- CSS
- JavaScript

## Purpose
This project was created to practice front-end web development and build a professional marketing website for a supplement store.
